## Simple Electron Lazy Issue

This is a simple Electron Lazy Issue project just to show an issue

### Install

#### Clone this repo

```
git clone https://github.com/isalo/electron-lazy-issue.git
```

#### Install dependencies

```
npm install
```

or

```
yarn
```

### Usage

#### Run the app

```
npm run start
```

or

```
yarn start
```

#### Build the app (automatic)

```
npm run package
```

or

```
yarn package
```

#### Build the app (manual)

```
npm run build
```

or

```
yarn build
```

#### Test the app (after `npm run build` || `yarn run build`)

```
npm run prod
```

```
yarn prod
```
